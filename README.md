# 🌿 Medicinal Leaf Classification

## 📌 Giới thiệu
Dự án sử dụng Deep Learning để phân loại và nhận diện lá cây dược liệu.

## 🧠 Công nghệ sử dụng
- Python
- PyTorch / TensorFlow
- OpenCV
- YOLO / CNN

## 📊 Dataset
Dataset được lưu trên Kaggle:  
https://www.kaggle.com/datasets/khanhfin/data-leaves

## 🚀 Quy trình thực hiện
1. Thu thập dữ liệu
Sử dụng dataset từ Kaggle
2. Tiền xử lý dữ liệu
Resize ảnh, chuẩn hóa dữ liệu
Áp dụng data augmentation để tăng độ đa dạng
3. Huấn luyện mô hình
Sử dụng CNN / YOLO để xây dựng mô hình
4. Tối ưu tham số nhằm cải thiện hiệu suất
Đánh giá mô hình
5. Sử dụng các chỉ số như accuracy và loss
Đạt độ chính xác khoảng 86%
6. Định hướng phát triển
Triển khai thành API (FastAPI)
Xây dựng ứng dụng nhận diện thực tế

## 📷 Kết quả
- Accuracy: 86%
