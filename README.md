# 🌿 Medicinal Leaf Classification

## 📌 Giới thiệu
Dự án sử dụng Deep Learning để phân loại và nhận diện lá cây dược liệu.

## 🧠 Công nghệ sử dụng
- Python
- PyTorch / TensorFlow
- OpenCV
- YOLO / CNN

## 📊 Dataset
Dataset được lưu trên Kaggle:  
https://www.kaggle.com/datasets/khanhfin/data-leaves

## 🚀 Cách chạy
1. Cài thư viện:
pip install -r requirements.txt  

2. Chạy notebook hoặc file train

## 📷 Kết quả
- Accuracy: 86%
